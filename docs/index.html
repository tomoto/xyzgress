XYZGRESS

(Sorry for Japanese only, for now.)

【これは何？】

kotakuさんの手によるIngressをレトロゲーム化した「XXGRESS」のオマージュとして、私Tomotoが作った移植版です。

本家のXXGRESS、すごく面白いんですよ。面白くてしばらくハマってしまったのですが、やっているうちに「ここを変えたらもっと面白くなるはず！」とアイディアが次々出てきて、結局それらを盛り込んだ移植版を作っちゃいました。本家の素朴さは若干失われてしまいましたが、より奥深く、頭を使わされるゲームになったと思います。

(ゲーム画面が表示されるまで少々時間がかかるかもしれません)

【遊び方】

本家と同じく、
  • カーソルキー: 移動
  • [Z]: バースター発射
  • [X]: 近くのポータルにリンクを張る
あとはAGなら説明しなくてもわかってくれるハズ！

【ゲームの目的】

これも本家と同じ、7日目を迎えたときに相手より100MU以上多く支配していれば勝ちです。

【強化点のハイライト】

  • 難易度(敵の数)を選べます。
  • 敵が自分と同じバースターを撃ちます。
  • リンクによるポータル強化、水没、リチャ防衛、リアキャプ妨害、廃人撃ち(敵4人モードでのみ解禁)、スーパーノヴァ、…といった実際のIngressでおなじみの戦術が使えます。やり方や活用方法は自分で見つけてください！
  • Unity WebGLを使っているので重いかもしれません…
