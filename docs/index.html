XYZGRESSって何？

Ingressをレトロゲーム化した「XXGRESS」(作者:kotakuさん)のオマージュとして作った移植版です。

本家のXXGRESS、すごく面白いんですよ。面白くてしばらくハマってしまったのですが、わりとすぐに常勝パターンが固まってしまい、「ここを変えたら・直したらもっと奥深くなるはず！」とアイディアが次々出てきて、結局それらを盛り込んだ移植版を作っちゃいました。もとの素朴さは若干失われてしまいましたが、より頭を使わされるゲームになったと思います。

遊んでみよう！

下にゲーム画面が表示されます。カーソルキーと[Z][X]だけで遊べるのでレッツトライ！

※WebGLをサポートしたブラウザが必要です。動き出すまで時間がかかるので気長にお待ちください。

7日目を迎えたときに相手より100MU以上多く支配していれば勝ちとなります。

本家との主な違い

  • 難易度(敵の数)を選べます。
  • 敵が若干賢くなっており、自分と同じくバースターで攻撃してきます。
  • Ingressの仕様が追加で盛り込まれており、リンクによるポータル強化、CFによる沈没、リチャ防衛、リアキャプによる妨害、廃人撃ち(敵4人モードでのみ解禁)、ノヴァ作戦…といった実際のIngressでおなじみの戦術が使えるようになっています。やり方や活用方法は自分で見つけてください！
  • APとエージェントレベルが測定されます。MUを狙うか、メダルを狙うか、APを狙うかはあなた次第です。
  • 張れるはずのリンクが張れないことがあるバグが直っています(実はこれが一番対策したかった)。
  • 音はありません。 音がいろいろと出て、状況把握の役に立ちます。
