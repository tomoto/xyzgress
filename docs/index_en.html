What's XYZGRESS？

Recently one Ingress fan kotaku made Ingress into a retro-looking game called XXGRESS. Now, this XYZGRESS is my version of it as homage.

The original XXGRESS was REALLY great. I played it again and again, and eventually got many ideas of improvements and fixes that would make the game more tactically interesting. I did not feel like just leaving those ideas, hence created this version of mine. While I pretty much liked the minimalism of the original one, I certainly believe you can have a different kind of fun to outsmart the tough computer agents with this version.

Getting started

The game shows up below. Arrow keys and [Z][X] are all you need to use, so please don't be shy and try it!

Note: WebGL-supported browser is required. It may take time to load the game, so please be patient if it looked unresponsive.

How to play

It is just like the original XXGRESS. I am sure you can get how it works quickly if you have some Ingress experience.

Cursor keys
Move
[Z]
Fire burster, select menu
[X]
Link to nearest portal
[ESC]
Withdraw

You will fight for 7 days, and win the game by controlling 100+ MUs more than the enermy in the end.

How this version (XYZGRESS) is different from the original (XXGRESS)?

  • You can choose the difficulty, i.e. number of enemies.
  • The enemies are smarter and use the same burster to attack your portals.
  • More Ingress features are implemented and you can use a variety of common Ingress tactics to win the game, e.g. links to increase the portal's damage mitigation, CF cover to restrict the enemy moves, recharging to defend important portals, saying hi to enemies to slow them down, rapid-firing the bursters (only unlocked in 4-enemy mode), nova tactics, etc. Find yourself how you can execute and leverage these tactics.
  • AP and the agent level are measured.
  • One frustrating bug, where sometimes links are not created when you theoretically can, was fixed.
